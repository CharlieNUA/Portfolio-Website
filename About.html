About Me

Hello I'm Charlie, and I'm a upcoming Games Developer!

Based in the uk, I'm currently in my third year of university at NUA (Norwich University of the Arts) studying Games Development to join the Games industry when i finish my learning and get a degree. My main intrest is how games work and how you can effect it by changing small details, this had followed me with technology in general as i enjoy learning about new software, engines, components, peripherals and more. Through my course i have been able to pick up new games design techniques and work on projects that add to my game libary and solidify working as a team to create something realy special.

My main engine i use to create games is unreal engine 4, which i have spent a good amount of time understanding what all the different features and functions do.While im well versed in blueprint i prefer using rider or visual studio to create the code that runs the game because you have so much more fredom and customisabilty to create what you want, which im still buidling apon at university.My languages i know are HTML, Javascript, C++ and Css which i have a decent understanding of how they work however i still have a keen instrest in learning new coding languages since they can be a valuable asset to a game developers arsonal.

Outside of my studying i have been able to work with other students and people in the industry to create games for game jams and events which have been very fun and i enjoy competeing in them, not only for my own games but seeing what other groups have done and playtesting there games to see what works and what doesn't.

I really enjoy watching and playing video games: some new and some old that give me inspiration to create something of my own with my own look or idea. Ive been playing games since i was aleast 6 and have picked up many new genres and franchies along the way, that shape what i like today.
